How to add the encrypted video to Unity Stream

    

      

        
1

        

          Encrypt here. Drop your video, click "Encrypt & Download .bin".
          A .bin file is saved to your Downloads.
        

      

      

        
2

        

          Place in assets folder. Move the .bin file into your project's
          assets/ folder (Vite serves it at /filename.bin).
        

      

      

        
3

        

          Register in videos.js. Copy the generated snippet shown after encryption
          and paste it inside the VIDEOS array in src/data/videos.js.
        

      

      

        
4

        

          Run the dev server. npm run dev — the new video appears in
          the library. The player will fetch, decrypt, and stream it automatically.
